# 🎹 KeyScape – Animated Multi-Instrument Web Piano

An interactive 3D web-based piano keyboard with realistic sound, smooth animations, recording capability, and multiple instruments.

## ✨ Features

- 🎹 2 Octave Responsive Keyboard
- 🎛 Instrument Selector (Piano, Organ, Synth, Electric)
- 🎧 Polyphonic Sound Support
- 🎬 Falling Notes Animation
- 🌊 Audio Waveform Visualization
- ✨ Particle Effects & Ripple Animations
- ⏺ Recording & Playback
- 🎼 Chord Detection Display
- 🎚 Volume Control
- 🎵 Sustain Pedal (Spacebar)
- 🌈 Theme Modes (Classic, Neon, Minimal)
- 📱 Fully Responsive (Desktop / Tablet / Mobile)

## 🎮 Controls

White Keys: A - L  
Black Keys: W - U  
Spacebar: Sustain  
Arrow Keys: Octave Shift
